# Consent Manager SDK Web

## Getting started

### Setup

`npm install consent-manager-js-sdk`

or

`yarn add consent-manager-js-sdk`

Copy-paste the following lines into the `head` of your `index.html` :

```
<style>
    .cmjs-form {
    height: 80%;
    width: 80%;
    top: 10%;
    left: 10%;
    position: fixed;
    }
</style>
```

### Methods

`ask(): affiche une alerte dans le navigateur`
